树状数组区间加区间求和

假设原数组是{ai​}，差分数组{di​=ai​−ai−1​}

易知ax​=∑i=1x​di​

将区间和拆成两个前缀和之差后，可得
Ans=∑i=1n​ai​=∑i=1n​di​(x−i+1)
Ans=∑i=1n​(n+1)di​−∑i=1n​idi​

用树状数组维护di​与idi​的前缀和即可，每次区间加仅会改变l与r+1处的di​与idi​的值，这一部分对应树状数组的单点加，求di​与idi​的前缀和则对应树状数组的前缀求和。

struct BIT{
	ll a[N],b[N];
	
	void add(int x,ll d){
		int t=x;
		for(;x<=n;x+=x&-x){
			a[x]+=d;
			b[x]+=d*t;
		}
	}
	
	void add(int l,int r,ll d){
		add(l,d);
		add(r+1,-d);
	}
	
	ll sum(int x){
		ll ret=0;
		int t=x+1;
		for(;x;x-=x&-x){
			ret+=a[x]*t;
			ret-=b[x];
		}
		return ret;
	}
	
	ll sum(int l,int r){
		return sum(r)-sum(l-1);
	}
}T;
